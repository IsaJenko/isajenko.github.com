1.$cd

命令cd表示：进入某个文件夹目录，或返回某目录

  • cd / 进入根目录
  • cd bin 进入某目录(如：进入bin目录)
  • cd .. 返回上层目录
  • cd ../ ../ 返回上上层目录(中间有没有空格都一样)
  • cd ~ 进入用户目录

2.$ls

命令ls表示：显示本层目录的所有文件和文件夹

$ ls
当ls后面加上具体的目录时直接输出该文件夹得详细目录信息(比如bin目录下)

$ ls bin

3.命令记忆功能

按上下键，可以查看使用过的所有命令。

4.自动补全功能

  • 按tab键，当在一串指令的第一个词后面，自动补全命令
  • 按tab键，当在一串指令的第二个词后面，自动补全文件名或文件目录
  • 连按两次tab键，(上面两种晴空中，若再按一次tab键)，则显示所有某些字母开头的命令，或本目录下的所有文件名。

5.关机命令

sudo shutdown -h now

有些时候系统出问题关机不成功可以试试用命令行进行关机，毕竟裸关电源不好。

6.如何通过终端命令删除Finder中无法删除的文件

有些时候因为某些原因，比如死机、文件下载到一半意外退出，这时经常有文件无法删除，系统提示你权限不够。这个时候我们就可以利用一行简单的命令进行删除。

1).打开终端应用程序

2).输入命令：sudo rm -r -f"你要删除的文件"

还要注意终端命令式区分大小写的，这里全部是小写。

3).把你要删除的文件或者文件夹用鼠标拖进终端窗口，也可以一次拖好多进来

4).然后再终端中回车

5).输入当前管理员用户密码。如果没有密码就直接回车。

6).终端中没有任何提示信息表明成功删除

什么都不显示才是好命令

注意：

如果用这个命令还无法删除，有两种可能性：一是你删除的是系统正在使用的临时文件之类的。二是有可能你的硬盘目录出现问题。这时清先用磁盘工具检查一下你的硬盘。

解释：

sudo命令：任何命令前都可以加上sudo，sudo表示是管理员权限，可直接管理系统中较重要的文件，一般使用需要询问密码。输入管理员密码后，才能完成操作。
