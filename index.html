XActivatePowerMode

An Xcode plug-in makes activate-power-mode happen in your Xcode.

NOTE THAT THIS VERSION IS VERY BUGGY RIGHT NOW

Effects

  • Default

  • Bloody (Thanks to @gavinkwoe)

  • 🎉 Find More

How to install and use?

Terminal

Alcatraz

Install Alcatraz followed by the instruction (search XActivatePowerMode), restart your Xcode and it works by default.

Manually

Clone the repo, then build (cmd + b) the XActivatePowerMode target in the Xcode project and the plug-in will automatically be installed in ~/Library/Application Support/Developer/Shared/Xcode/Plug-ins. Relaunch Xcode and it will work too.

Deactivate Tips

The switch is under the menu Edit -> Activate Power Mode, just click that or uninstall this plug-in :] .

Make your own particle effect?

The default effect was builded with UIEffectDesigner. Make your own ped effect file, then add it to the project. The name of it will show in the effects menu list.

Its appreciated that you pull a request with your EFFECT.

By Jenko

发表于 2015-11-20

The Noob Guide(新手指导)

原文地址

首先,也是最重要的,重要的是你明白“黑客”是一个宽泛的术语。有许多方面,其中大部分需要一些编程技巧,但不应该阻止你去使用互联网上可用的工具。

去HTMLdog网站学习一些HTML,它是一个非常棒的网站,你将会从中学习并进步。此外,请考虑Python作为你的第一编程语言;因为它是最根本的和适当的语言,所以我推荐新人去学习并使用，会让你走进计算机的世界。现在你带着问题已经准备好继续了吗,请允许我给你一个简单的参考指南。

从一个方面说明,在开始之前,确保你的网络连接有某种形式的保护,通过代理或VPN。不要采取任何其他匿名成员的建议,甚至行动。他们可能试图通过你的vpn服务与联邦政府合作。在你的VPN上做你自己的调查,隐私政策,协议条款等。没有人会为你进监狱。之后,用匿名支付方式,再次支付,用其他的方法调查。

(PS:不要在VPN进行DDoS攻击。他们可能从目的地保护你的数据,但尽量不要去信任他们当你发送没完没了的数据包在他们的服务器上。不用说,你的安全对我来说是没有任何关心的,所以首先照顾好自己。)
